# NhorquiMaq - Página web

Página web informativa para la empresa NhorquiMaq, empresa que presta servicios de grúas horquillas hacia el sector portuario de Iquique.

---

## ¿Qué hace?

- **Información de la Empresa**, como lo es su misión y visión, además de la descripción general de esta misma.
- **Servicios prestados**, la diversidad de servicios que presta hacia sector portuario, zona franca, etc.
- **Clientes** que han confiado en los servicios de la empresa, como por ejemplo EPI.
- **Flota de grúas** con los que cuenta la empresa, desde grúas de 3T hasta 5T

---

## Stack y Arquitectura

| Capa                 | Tecnología                                                                            |
| -------------------- | ------------------------------------------------------------------------------------- |
| Frontend             | HTML5, CSS3, JavaScript                                                               |
| Framework UI         | Bootstrap 5.3.3, Bootstrap Icons                                                      |
| Estilos              | CSS personalizado en `style.css`                                                      |
| Interactividad       | JavaScript propio en `script.js`                                                      |
| Formularios          | FormSubmit para envío de mensajes por correo                                          |
